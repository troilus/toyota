注意事项

本手册未包括车辆维修和维护所必需的全部项目。 仅供具有专业技术资格的人员使用。 如果非专业人员或未取得专业资格证书的人员仅仅参照本手册执行维修或维护，或者没有使用专用维修设备或工具，可能会导致自身或周围人员严重受伤和客户车辆的损坏。

为了防止危险操作，避免客户车辆损坏，请务必遵守以下说明。
  • 必须通读本手册内容。 尤其要充分理解“导言”部分中“注意事项”的所有内容。
  • 本手册中提供的维修方法对于车辆修理和维护非常有效。 在遵循本手册中的程序进行维修操作时，务必使用规定或推荐的工具。 如果不使用规定或推荐的工具和维修方法，则在开始工作前务必确保技术人员的安全，且不会造成人身伤害或客户车辆损坏。
  • 如果需要更换零件，则必须换上具有相同零件号的零件或同等零件。 不要采用劣质零件。
  • 值得注意的是，为了有效避免维修或维护期间可能造成的人身伤害，以及由于操作不当而造成的车辆损坏和隐患，必须认真遵守本手册中各种“注意事项”和“备注”事项。 还应该注意的是，本手册中的“注意事项”和“备注”部分的内容并非夸大其词。 因此，要充分认识到违反这些说明有可能导致危险的后果。
