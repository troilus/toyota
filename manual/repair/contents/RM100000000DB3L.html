防盗/无钥匙上车系统  停机系统（带智能进入和起动系统）  B2790  ID 代码箱 EEPROM 故障  


描述

ID 代码箱（停机系统代码 ECU）出现内部故障时，认证 ECU（智能钥匙 ECU 总成）存储该 DTC。

DTC 编号 检测项目 DTC 检测条件 故障部位 备注
B2790 ID 代码箱 EEPROM 故障 ID 代码箱（停机系统代码 ECU）出现内部故障（EEPROM 读取故障）（单程检测逻辑*）。
·

ID 代码箱（停机系统代码 ECU）

·

认证 ECU（智能钥匙 ECU 总成）


DTC 输出确认操作：

进行停机系统设定/解除操作（携带电子钥匙发射器分总成时将发动机开关置于 OFF 或 ON (IG) 位置）。



*：仅存在故障时输出。


检测到故障时的车辆状况和失效保护操作
检测到故障时的车辆状况 检测到故障时的失效保护操作
·

发动机运转：发动机下次将无法起动

·

发动机停止：发动机无法起动

注意/小心/提示

备注：
·

发动机开关置于 OFF 位置的情况下使用 GTS 时，将 GTS 连接至 DLC3 并以 1.5 秒或更短的间隔打开并关闭门控灯开关，直至 GTS 和车辆之间开始通信。然后在手动模式下选择车型代码“KEY REGIST”并进入以下菜单：Body Electrical / Entry&Start(CAN)。使用 GTS 时，以 1.5 秒或更短的间隔间歇打开并关闭门控灯开关，以保持 GTS 和车辆间的通信。

·

更换认证 ECU（智能钥匙 ECU 总成）或 ID 代码箱（停机系统代码 ECU）前，请参考《服务快讯》。

·

维修后，通过进行“DTC 输出确认操作”确认未输出 DTC。
