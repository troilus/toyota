RM3140CG _51 发动机/混合动力系统 _023966 4ZR-FE 发动机控制系统 _0129818 凸轮轴机油控制阀 G

4ZR-FE 发动机控制系统  凸轮轴机油控制阀  车上检查  


程序

1.检查凸轮轴正时机油控制阀总成

a.

将 GTS 连接到 DLC3。


b.

起动发动机。


c.

打开 GTS。


d.

打开空调开关。


e.

检查凸轮轴正时机油控制阀总成（进气侧）。

i.

进入以下菜单：Powertrain / Engine and ECT / Active Test / Control the VVT Linear (Bank1)。

Powertrain > Engine and ECT > Active Test
检测仪显示
Control the VVT Linear (Bank1)
50004 234 32

ii.

使用 GTS 操作凸轮轴正时机油控制阀总成时，检查发动机转速。

标准状态:
GTS 操作 规定状态
0% 发动机转速正常
100% 发动机怠速不稳或熄火


提示：
·

如果结果不符合规定，则冷却发动机（发动机冷却液温度为 50°C (122°F) 或更低）并再次进行主动测试。

·

可能因主动测试操作存储与 VVT 系统相关的 DTC，但这并不是故障。



如果操作不符合规定，则检查凸轮轴正时机油控制阀总成、线束和 ECM。



f.

检查凸轮轴正时机油控制阀总成（排气侧）。

i.

进入以下菜单：Powertrain / Engine and ECT / Active Test / Control the VVT Exhaust Linear (Bank1)。

Powertrain > Engine and ECT > Active Test
检测仪显示
Control the VVT Exhaust Linear (Bank1)
50004 234 87

ii.

使用 GTS 操作凸轮轴正时机油控制阀总成时，检查发动机转速。

标准状态:
GTS 操作 规定状态
0% 发动机转速正常
100% 发动机怠速不稳或熄火


提示：
·

如果结果不符合规定，则冷却发动机（发动机冷却液温度为 50°C (122°F) 或更低）并再次进行主动测试。

·

可能因主动测试操作存储与 VVT 系统相关的 DTC，但这并不是故障。



如果操作不符合规定，则检查凸轮轴正时机油控制阀总成、线束和 ECM。
